The loop

Three unbroken steps, from a running timer to money in the bank.

Everything off this loop is deferred. FluxWork does one thing completely.

01

Track time

Start a timer or add it manually. Flag each task billable or non-billable — the split is first-class, not an afterthought.

02

Generate the invoice

Billable, un-invoiced time fills the built-in template. Rates are snapshotted, one tax line applied, exported DOCX → PDF.

03

Get paid

Send the PDF, mark it invoiced. Subscriptions run through Paddle as merchant of record — tax handled, payouts sorted.

Why it’s different

The billable-to-invoice loop, closed with financial precision.

Billable / non-billable split

Every task carries a billable flag that flows into earnings and invoices. Non-billable time is tracked but never billed by accident.
